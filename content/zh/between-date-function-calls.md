## 1 导言

在日期间函数调用计算两个日期之间的差额。

## 2 毫秒之间

`毫秒之间` 函数计算输入值之间以毫秒为单位的差值。

### 2.1 输入参数

下面的表格描述了输入参数：

| 值     | 类型    |
| ----- | ----- |
| 第一个日期 | 日期和时间 |
| 第二个日期 | 日期和时间 |

### 2.2 产出

产出情况见下表：

| 值                  | 类型 |
| ------------------ | -- |
| 以毫秒为单位测量的两个日期之间的差异 | 小数 |

### 2.3 例子

如果您使用以下输入：

```java
millisecondsBetween(dateTime(2007, 1, 1, 1, 1, 1, 1, 1), dateTime(2007, 1,1,1,3))
```

输出为：

```java
2000
```
