汇总对检索栏数值进行具体计算。 以下汇总函数是可能的：

| 平均值 | 平均值 |
| --- | --- |
| 币种  | 计数  |
| 最大值 | 最大值 |
| MIN | 最小值 |
| SUM | sum |

当您在SELECT 条款中使用汇总表达式时， SELECT 条款中的所有表述都必须是查询中的一个汇总或一部分。
