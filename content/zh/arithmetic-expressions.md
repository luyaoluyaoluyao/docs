## 1 导言

本文档描述了表示支持的算术运营商。 所有这些都适用于数字数据类型(整数/长/小数)。

## 2 乘法

乘以两个数字。

### 2.1 输入参数

下面的表格描述了输入参数：

| 值     | 类型    |
| ----- | ----- |
| 第一个数字 | 整数/小数 |
| 第二个数字 | 整数/小数 |

### 2.2 产出

如果这两种输入都是整数/Long，结果是整数/Long。

如果两个输入中的任何一个都是小数型，结果是小数型。

### 2.3 例子

如果您使用以下输入：


```java
2*3
```
输出为：

```java
6
```

## 3 Division

除去两个数字。 您可以使用 `div` 或 colon ( `:` ) 语法, 如下面的示例所示。 冒号 ( `:` ) 语法是由分隔符号 `X` 启发的。 我们不能使用更常规的 slash ( / ) 语法，因为这会与用于分离对象和成员的斜线相冲突。

### 3.1 输入参数

下面的表格描述了输入参数：

| 值     | 类型    |
| ----- | ----- |
| 第一个数字 | 整数/小数 |
| 第二个数字 | 整数/小数 |

### 3.2 产出

如果两个输入中的任何一个都是小数型，结果是小数型。

### 3.3 示例

查找下面的用法示例：

* `div` 语法示例：如果您使用了以下输入：

  ```java
  3 div 5
  ```

  输出为：

  ```java
  0.6
  ```

* `:` 语法示例: 如果你使用以下输入:

  ```java
  12 : 3
  ```

  输出为：

  ```java
  4.0
  ```

### 3.4 评论

分割的结果只是一个近似值，如果它有无限的小数点扩张。 以下两个例子说明了这种近似情况：

* 如果您使用以下输入：

    ```java
    3 : 7
    ```

    输出为：

    ```java
    0.4285714285714285714285714285714286
    ```

    如果你继续计算一个分区的结果，结果可能是意料之外的。 以下输入：

    ```java
    (3 : 7) * 7
    ```

    以下输出结果：

    ```java
    3.0000000000000000000000000000000002
    ```

* 如果您使用以下输入：

    ```java
    最高限额(3：7*7)
    ```

    输出为：

    ```java
    4
    ```

因此，建议在最后完成各司的业务。

## 4 Modulo

### 4.1 输入参数

下面的表格描述了输入参数：

| 值     | 类型    |
| ----- | ----- |
| 第一个数字 | 整数/小数 |
| 第二个数字 | 整数/小数 |

### 4.2 产出

如果这两种输入都是整数/Long，结果是整数/Long。

如果两个输入中的任何一个都是小数型，结果是小数型。

### 4.3 例子

如果您使用以下输入：

```java
23 mod 5
```

输出为：

```java
3
```
