## 1 导言

这些职能是将日期轮到不同的时间单位。

## 2 trimToSeconds

此函数用于将日期分割为秒，将所有毫秒四舍五入为零。

### 2.1 输入参数

下面的表格描述了输入参数：

| 值        | 类型    |
| -------- | ----- |
| 应该修剪的日期。 | 日期和时间 |

### 2.2 产出

产出情况见下表：

| 值               | 类型    |
| --------------- | ----- |
| 同一日期，但是四舍五入到秒数。 | 日期和时间 |

### 2.3 例子

例如，您有一个变量 `$myDate` 表示'08-06-2008:12:51.387'：

```java
trimTosec($myDate)
```

输出为：

```java
'Sun 08 10:12:51 CEST 2008'
```
