参考集选择器允许您通过选择对象来设置类型引用组的关联。 例如，如果客户可以属于几个组别，可以使用参考集选择器来选择客户所属的组。 这就要求在域模型中设置一个从客户到类型参考组的关联。

{{% alert type="info" %}}

![](attachments/pages/reference-set-selector.png) 在这个客户数据视图中，您可以使用参考集选择器选择多个组。

![](attachments/pages/reference-set-selector-domain-model.png) 在域模型中有一个从客户到群组的类型参考集合。

{{% /报警 %}}

参考集选择器看起来很像一个 [数据网格](data-grid) ，因此与数据网格共享许多属性。 主要差异是参考集选择器缺少搜索栏，默认情况下有不同的按钮(添加、移除)。

{{% alert type="warning" %}}

**添加** and **删除** 按钮意味着除 **新的** 之外的东西， **删除** 默认在数据网格中使用的按钮：

*   **添加** 按钮可添加引用到一个现有对象。 另一方面， **创建** 按钮，创建一个新对象，不添加引用。
*   **删除** 按钮实际上删除了一个对象好的。 而 **删除** 按钮只是删除对对象的引用并使对象本身保持不变。

{{% /报警 %}}

## 外观属性

### 显示控制栏

此属性表示控制栏是否在终端用户界面中可见。 控制栏还包括分页按钮。

{{% alert type="warning" %}}

即使控制栏不可见，但仍然可能有一个默认按钮，它是由单击一行触发的 (double) 。 查看属性“默认按钮触发器”和“ [控制条](control-bar) 以获取更多信息。

{{% /报警 %}}

_默认值：_ True

### 显示分页按钮

此属性表示通过网格信息页面的按钮是否可见。 只有当您确定不会有比网格行数更多的对象时，才会隐藏这些按钮。 请注意，隐藏控制栏也隐藏分页按钮。

_默认值：_ True

### 列权重

列重量是以半冒号隔开的百分比。 它们决定栏的宽度。 重量必须增加到 100\。 改变列宽度的另一种方法是拖动列之间的分隔线。

{{% alert type="info" %}}

列权重在上方的截图中为63;37。

{{% /报警 %}}

### 行数

使用此属性您可以更改将显示在一个页面中的行数。 另见属性“显示空行”。

_默认值：_ 5

### 显示空行

如果您选择显示空行，即使在页面上显示的对象较少，但总是有显示相同行数的格子(见“行数”)。

_默认值：_ False

## 行为属性

### 选择模式

选择模式决定用户是否以及如何在网格中选择项目。

| 值     | 描述                                                                     |
| ----- | ---------------------------------------------------------------------- |
| 没有选择  | 用户不能选择项目。                                                              |
| 单选    | 用户可以通过点击它选择单个项目。 点击另一个项目将使该项目成为选择。                                     |
| 多选    | 用户可以通过点击第一个项目选择多个项目并按住“Ctrl”键，同时点击其他项目。 只需单击项目就可以取消选择所有项目并使单击项目成为选中项目。 |
| 简单的多选 | 用户可以只需单击即可选择多个项目。 这是版本3.1.0之前默认的多选行为。                                  |

_默认值：_ 单个选择

### 默认按钮触发器

默认按钮可以由单击或双击单行触发。

| 值    | 描述                           |
| ---- | ---------------------------- |
| 单次点击 | 单击触发默认按钮。 此选项不能与允许用户选择行一起使用。 |
| 双击   | 双击触发默认按钮。                    |

_默认值：_ 双击一次

### 工具提示页面

工具提示页面是在鼠标悬停在一行时出现的页面。 工具提示页面应包含与数据网格相同的实体上的数据视图。 在创建和连接的工具提示页面上，您还必须指定工具提示会出现在哪个列上。 查看 [数据网格列的 '显示工具提示' 属性](columns)。

## 数据源属性

### 实体路径

参考集选择器的实体路径是从含有数据视图的实体向该协会另一侧的实体沿着一个类型的关联“参考集”的路径。 在上面的例子中，客户到客户之间的关联组被遵循，路径是：Customer_Group/Group。

{{% alert type="warning" %}}

使用参考选择器您正在编辑关联类型参考集(Customer_Group)。 添加和删除使用选择器的对象只会添加和删除这些对象的引用；对象本身没有变化。

{{% /报警 %}}

{{% alert type="warning" %}}

尚不支持在参考集选择器中使用不可持续的实体 (NPEs)。 因此，在实体选择器中为参考集选择器选择一个NPE是不可能的。

{{% /报警 %}}
