---
title: "创建日期"
parent: "表达式"
---

可以使用 [解析日期时间](parse-and-format-date-function-calls) 创建某个日期的日期类型变量。 这需要一个日期字符串和一个格式字符串作为参数，并返回一个日期类型变量。 欲了解更多详情，请参阅 [parseDateTime](parse-and-format-date-function-calls)。

可以使用 `日期时间` 和 `日期时间` 函数创建代表日期的字符串变量。 这两个函数之间的差别是 `日期时间` 使用函数调用中使用的会话日历， and `dateTimeUTC` 使用的 UTC 日历。 系统会话默认以 UTC 方式运行，但可以在 [项目设置](project-settings) 中进行配置。

这些函数需要 1 到 6 个输入参数并返回一个字符串。 这些代表的顺序是：

1. 年
    * 类型：整数、四位数字和大于 1799
2. 月
    * 类型: 整数, 介于 1 到 12 之间
3. 天
    * 类型：整数介于 1 到 31 之间
4. 小时数
    * 类型：整数，0至23
5. 分钟
    * 类型：整数，0和59
6. 秒
     * 类型：整数，0和59

一个参数：

```java
日期时间(2007年)
```

返回：

```java
"Mon Jan 01 00:00:00 CET 2007"
```

两个参数：

```java
日期时间(2007年，1)
```

返回：

```java
"Mon Jan 01 00:00:00 CET 2007"
```

三个参数：

```java
日期时间(2007年，1，1)
```

返回：

```java
"Mon Jan 01 00:00:00 CET 2007"
```

四个参数：

```java
日期时间(2007年，1，1，1)
```

返回：

```java
"Mon Jan 01 01:00:00 CET 2007"
```

五个参数：

```java
日期时间(2007, 1, 1, 1)
```

返回：

```java
"Mon Jan 01 01:01:00 CET 2007"
```

六个参数：

```java
日期时间(2007年，1，1，1，1，1，1，1，1，1)
```

返回：

```java
"Mon Jan 01 01:01:01 CET 2007"
```
