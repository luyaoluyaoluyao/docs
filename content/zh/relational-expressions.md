## 1 导言

关联表达式允许用户比较值并根据该信息进行更改和操作。 这种表达式的返回类型总是布尔值。

## 少于2( <)

此表达式决定第一个值是否小于第二个值。

### 2.1 输入参数

可能的数值见下表：

| 值   | 类型            |
| --- | ------------- |
| 两个值 | 字符串           |
| 两个值 | 数字(整数/中, 十进制) |
| 两个值 | 日期和时间         |

{{% alert type="info" %}}
两个数值应属于同一类别（例如，如果一个是数字，另一个则是数字）。
{{% /报警 %}}

### 2.2 产出

产出情况见下表：

| 值            | 类型      |
| ------------ | ------- |
| `是真` 或 `错误`。 | Boolean |

### 2.3 例子

如果您使用以下输入：

```java
4<3
```

输出为：

```java
错误
```

## 6 等于( = )

这个表达式决定这两个值是否相同。

### 6.1 输入参数

可能的数值见下表：

| 值   | 类型               |
| --- | ---------------- |
| 两个值 | 字符串              |
| 两个值 | 数字(整数/中, 十进制)    |

### 6.3 实例

下面的例子说明表达式返回的价值：

* 如果您使用以下输入：

    ```java

    ```

    输出为：

    ```java

    ```

* 如果您使用以下输入：


