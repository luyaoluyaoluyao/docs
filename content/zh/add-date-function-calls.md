将时间单位添加到一个日期。 在所有例子中，第一个输入可以是一个新的日期时间（在所有例子中描述）， 一个类型日期时间的变量，或者域实体类型日期时间的属性。

## addMillisecond

将几毫秒添加到一个日期。

### 输入参数

*   初始日期 类型：日期时间
*   要添加的毫秒数量 类型：整数

### 产出

与初始日期对应的类型日期加上x毫秒的结果。

{{% alert type="info" %}}

```java
addMilliseconds(dateTime(2007, 1, 1, 1, 1, 1, 1), 1400)

```

在对应的日期时间内结果

```java
"Mon Jan 01 01:01:02:400 CET 2007"

```

{{% /报警 %}}

## addDays[UTC]

将几天添加到一个日期。 `添加` 天数使用服务器的日历和 `添加 DaysUTC` 使用的 UTC 日历。

### 输入参数

*   初始日期 类型：日期时间
*   要添加的天数 类型：整数

### 产出

与初始日期对应的类型日期加上X天的结果。

{{% alert type="info" %}}

```java
addDays(dateTime(2007, 1, 1, 1, 1, 1, 1), 3)

```

在对应的日期时间内结果

```java
"Mon Jan 04 01:01:01 CET 2007"

```

{{% /报警 %}}

{{% /警示%}}!{% alert type="warning" %}}


