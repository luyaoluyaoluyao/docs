{{% alert type="info" %}}
この活性は、ナノフローではなく、マイクロフローでのみ使用できます。
{{% /alert %}}

## 1つの紹介

generate-document microflow アクションを使用して、 [テンプレート](document-templates) に基づいてドキュメントを作成できます。

{{% alert type="info" %}}

![](attachments/819203/918200.png) ドキュメントを生成

{{% /alert %}}{{% alert type="info" %}}

すべてのアクティビティが共有するプロパティ(図表など)については、 [Microflow Element Common Properties](microflow-element-common-properties) を参照してください。 このページでは、アクションに固有のプロパティのみを記述します。

{{% /alert %}}

## 2つの入力プロパティ

### 2.1 ファイルドキュメント

生成されたドキュメントを含むファイルドキュメント。 エンティティSystem.FileDocument のオブジェクトまたはそれの専門化である必要があります。

## 3 アクションプロパティ

### 3.1 言語

ドキュメントのタイトルとラベルが表示される言語。

| Option       | 説明                                                     |
| ------------ | ------------------------------------------------------ |
| 現在のユーザー      | 現在のユーザーの言語を使用します。                                      |
| プロジェクトのデフォルト | [プロジェクト設定](project-settings) で指定されたデフォルト言語を使用します。      |
| 変数           | 選択した変数に保存されている言語を使用します。 変数はSystem.Language型でなければなりません。 |

_デフォルト値:_ 現在のユーザー

### 3.2 ドキュメントタイプ

ドキュメント タイプは、生成されたドキュメントの種類を指定します。

| Option    | 説明                                 |
| --------- | ---------------------------------- |
| HTML      | HTML 形式でドキュメントを生成します。              |
| PDF       | PDF 形式のドキュメントを生成します。               |
| Word 2007 | Word 2007形式でドキュメントを生成します。          |
| Word 2003 | Word 2003形式のドキュメントを生成します。          |
| リッチテキスト形式 | リッチテキスト形式でドキュメントを生成します。            |
| ODT       | Open Office (ODT) 形式でドキュメントを生成します。 |

### 3.3 テンプレート

テンプレートは、 [ドキュメントテンプレート](document-templates) をファイルの生成に使用するものを定義します。 使用するテンプレートに応じて、1つ以上の引数を指定する必要があります。 トップレベルのデータグリッドと正しいタイプのデータビューオブジェクトごとに、テンプレートに渡す必要があります。 引数の値は式を使用して入力できます。

### 3.4 余白を上書き

ここでは、実行時に定義できるドキュメントのカスタムマージンを設定できます。
