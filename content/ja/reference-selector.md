リファレンスセレクタは、 [関連](input-widgets) を表示および編集するために使用できる [入力ウィジェット](associations) です。

{{% alert type="info" %}}

![](attachments/pages/reference-selector.png) このリファレンスセレクターでは、ご注文の商品を選択できます。 ![](attachments/pages/reference-selector-domain-model.png) 参照セレクタを使用するには、ドメインモデルで多対一の関連付けが必要です。

{{% /alert %}}

リファレンスセレクタは [レポート パラメータ](report-parameter) に似ていますが、ユーザが列挙にリストされている項目ではなく、関連付けを埋めるためのオブジェクトのリストから選択することができます。 セレクターでは、親オブジェクトが参照を共有するエンティティを選択する必要があります。 また、セレクターに表示したい参照エンティティからどの属性を選択する必要があります。

{{% alert type="success" %}}

_情報だけを_ 表示したい場合は、テキストボックスを使用することもできます。 これには、複数の関連付けステップに従うことができる追加の利点があります。

{{% /alert %}}

## 一般プロパティ

### 使用方法を選択

参照セレクタでは、ドロップダウンまたはポップアップページを使用してオブジェクトを選択できます。 ページを使用して選択する場合。 ドロップダウン機能は、選択ポップアップを開くウィジェットの右側のボタンに置き換えられます。

| 値       | 説明                      |
| ------- | ----------------------- |
| ページ     | ポップアップページを使用して参照を選択します。 |
| ドロップダウン | ドロップダウンを使用して参照を選択します。   |

{{% alert type="success" %}}

ドロップダウンを使用して選択することの利点は、非常に効率的であることです。ページを開く必要はありません。 ページを使用して選択する利点は、エンドユーザーがオブジェクトを検索できることです。 選択すべきオブジェクトがたくさんある場合(例えば20以上)、ページを使って選択することをお勧めします。

{{% /alert %}}

_デフォルト値:_ ドロップダウン

### オプションの図表番号を空にする

このプロパティは、ユーザーに表示されるドロップダウンの参照セレクタの空のオプションのキャプションを表します。 format@@0オプションにformat@@1プロパティが設定されている場合、キャプションの設定は使用できません。 これは翻訳可能なテキストです。 詳細については、 [翻訳可能なテキスト](translatable-texts) を参照してください。

{{% alert type="info" %}}

空のオプションにキャプションを記入すると、アプリケーションのユーザーエクスペリエンスが向上します。 また、スクリーンリーダーのユーザーがアプリケーションを簡単に操作するのに役立ちます。

{{% /alert %}}

{{% alert type="info" %}}

空のオプションキャプションは Mendix 7.2.0 から利用できます。

{{% /alert %}}

### ページを選択

format@@0 プロパティにより、ページの選択ボタンが使用されたときに開かれるページが決定されます。 このページは、可能なすべてのオブジェクトのリストから関連するオブジェクトを選択するために使用できます。 このページには、入力参照セットセレクタと同じエンティティに接続されたデータグリッド、テンプレートグリッド、またはリストビューが含まれている必要があります。

[ページを開く](opening-pages) を参照してください。 コンテンツ内の選択ページを開くことは禁止されています。

{{% alert type="success" %}}

ウィジェットを右クリックして「選択ページを生成...」を選択すると、新しいページを生成することができます。

{{% /alert %}}

### Go-To Page

go-toページでは、エンドユーザーが選択されているオブジェクトの詳細な概要にすばやくアクセスできます。 このプロパティは、ユーザーに表示するページを決定します。 このページには、参照セレクタによって選択されたものと同じエンティティを持つデータビューが含まれている必要があります。

### Go-To ページ設定

これらの設定では、ページの開き方を指定します。

詳細は [ページを開く](opening-pages) を参照してください。

### 日付の書式（日付と時刻の種類の属性のみ）

日付フォーマットは、参照セレクタが日付、時刻、日付、またはリンクされた属性のカスタムバリエーションを表示するかどうかを決定します。 日付および/または時刻がどのようにフォーマットされるかは、データを表示するユーザーのローカライズによって異なります。

_デフォルト値:_ 日付

### 選択可能なオブジェクトのプロパティ

カテゴリ 'Selectable オブジェクト' 内のプロパティは、エンドユーザーが選択できるオブジェクトを指定します。 選択可能なオブジェクトには、相互排他的な2つの方法があります。

1.  マイクロフローは、選択可能なオブジェクトのリストを返します。
2.  コンテキストメカニズムとそのメカニズムに影響を与えるプロパティを考慮して、オブジェクトのリストが自動的に決定されます。

### マイクロフロー

マイクロフローが選択されている場合、マイクロフローが呼び出され、参照セレクタが表示するオブジェクトのリストを計算します。 マイクロフローは、ドロップダウンを使用して選択を行う場合にのみ使用できます。

{{% alert type="warning" %}}

このカテゴリの他のプロパティは、マイクロフローを使用するときに効果がないことに注意してください!

{{% /alert %}}

### マイクロフローの設定

マイクロフローの設定では、マイクロフローに渡すパラメーターを指定できます。

### XPath 制約

XPath 制約を使用すると、選択することのできるオブジェクトのリストを制限する手動制約を追加することができます。

{{% alert type="info" %}}

商品の参照セレクタ上の XPath 制約 `[InStock = true()]` は、在庫にある商品のみが選択可能であることを保証します。

{{% /alert %}}

このプロパティは、参照セレクタを埋めるためにマイクロフローを使用する場合には効果がありません。

### 制約を受けた人

参照セレクタは、1 つまたは複数のパスで制約することができます。 これは通常、ある参照セレクタを別のセレクタに依存させるために使用されます。 例えば、注文行を編集できるページでは、銘柄セレクタはカテゴリセレクタによって制約されます。 カテゴリを選択した後、商品セレクタはこのカテゴリに制約され、カテゴリ内の商品のみを表示します。

{{% alert type="info" %}}

![](attachments/819203/917938.png) _ドメインモデル_

ドメインモデルでは、注文行には、カテゴリと製品の両方へのタイプ参照の関連性があります。 この形式では、これらは2つの参照セレクタで編集できます。 3つ目の協会は、製品からカテゴリーまで、それら2つのエンティティの関係を説明しています。 このような「三角形」の領域モデルの一部が制約を可能にするものである。

![](attachments/16713884/16844014.jpg) _フォーム_

ページには2つの参照セレクタがあり、1つはカテゴリ、1つは製品です。 製品のためのものは、三角形を形成するドメインモデルを通るパスによって制限されます。

![](attachments/16713884/16844013.jpg)

{{% /alert %}}

このプロパティは、参照セレクタを埋めるためにマイクロフローを使用する場合には効果がありません。

### 並び順

ソート順は、参照セレクター内のアイテムが表示される順序を指定します。 複数の属性を並べ替えることができます (昇順と降順)。 ソート順が指定されていない場合、参照セレクタは displayed 属性をソートします。

_デフォルト値:_ ソート順なし

## データソースのプロパティ

### 属性パス

属性パスは、関連するエンティティのどの属性を参照セレクタに表示するかを指定します。 パスは、データビューのエンティティから始まるタイプ参照の関連付けに従う必要があります。

{{% alert type="warning" %}}

Order_Customer/Customer/Name は、エンドユーザーがユーザー名のリストからユーザーを選択できるようにします。

{{% /alert %}}{{% alert type="warning" %}}

属性を参照セレクタに接続しても、実際にオブジェクトを選択していることに注意してください。 この属性は、オブジェクトの可視化を提供するためのものです。 この点で、参照セレクタはチェック ボックスやテキスト ボックスなどの他のウィジェットとは異なります。 これらのウィジェットでは、属性の値を編集します。 参照セレクタでは、タイプ参照の関連付けを編集します。

{{% /alert %}}
