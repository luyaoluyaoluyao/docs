## 1つの概要

XML スキーマは XML スキーマ定義 (XSD) ファイル内で定義されており、モデル内でインポートすることができます。 特定の XML ドキュメントがどのように見えるかを記述します。 例を **図1** に示します。 スキーマは [マッピングのインポート](import-mappings) と [マッピングのエクスポート](export-mappings) で使用することができます。
