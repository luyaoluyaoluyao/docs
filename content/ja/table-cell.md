## 一般プロパティ

### セルの種類

セルの種類を示します。 2 つのタイプがサポートされています:

| 値    | 説明                                           |
| ---- | -------------------------------------------- |
| 標準   | データを含むノーマルセル (<td>).    |
| ヘッダー | キャプションを含むヘッダーセル (<th>). |

_デフォルト値:_ 標準
