アクティビティは、マイクロフローで実行されるアクションを表します。

アクティビティの種類は次のとおりです。

## オブジェクトの操作

オブジェクトのアクティビティはオブジェクトの作成や操作に使用できます。 [ドメイン モデル](domain-model) は、使用可能なオブジェクト タイプ ([エンティティ](entities)) を定義します。

| グラフィック                                                       | 名前                               | 説明                                                                                                                          |
| ------------------------------------------------------------ | -------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| [![](attachments/819203/918013.png)](cast-object)            | [オブジェクトのキャスト](cast-object)       | キャストオブジェクトは、 [継承分割](inheritance-split) と組み合わせてオブジェクトの [特別な](entities) メンバーを使用することができます。                                     |
| [![](attachments/819203/917661.png)](change-object)          | [オブジェクトの変更](change-object)       | Change object can be used to change the members of an object. これは、イベントの有無にかかわらず、コミットせずに行うことができます。                           |
| [![](attachments/16713769/17661961.png)](committing-objects) | [コミットオブジェクト](committing-objects) | コミットオブジェクトは、1つまたは複数のオブジェクトに変更をコミットするために使用できます。                                                                              |
| [![](attachments/819203/917756.png)](create-object)          | [オブジェクトを作成](create-object)       | Create object can be used to create an object.                                                                              |
| [![](attachments/819203/918191.png)](deleting-objects)       | [オブジェクトを削除](deleting-objects)    | オブジェクトの削除は、オブジェクトの削除に使用できます。                                                                                                |
| [![](attachments/819203/917866.png)](retrieve)               | [保留する](retrieve)                 | 取得は、別のオブジェクトの1つ(またはそれ以上の)関連オブジェクトを取得するために使用できます。 さらに、アクティビティはデータベースから直接1つ(またはそれ以上の)オブジェクトを取得することもできます。                      |
| [![](attachments/819203/918119.png)](rollback-object)        | [オブジェクトのロールバック](rollback-object) | ロールバックオブジェクトを使用すると、アクティビティより前のマイクロフローの部分でオブジェクトに対して行われた変更(コミットされていない)を取り消すことができます。 さらに、作成されたことがないがコミットされたことのないオブジェクトを削除します。 |

## リストのアクティビティ

リストアクティビティは、オブジェクトのリストの作成と操作に使用できます。

| グラフィック                                               | 名前                      | 説明                                                                     |
| ---------------------------------------------------- | ----------------------- | ---------------------------------------------------------------------- |
| [![](attachments/819203/918028.png)](aggregate-list) | [集計リスト](aggregate-list) | 集計リストは、最大値などの集計値を計算するために使用できます。 オブジェクトのリスト上のオブジェクトの最小、合計、平均、合計、合計の合計量。 |
| [![](attachments/819203/918007.png)](change-list)    | [リストの変更](change-list)   | 変更リストは、リスト変数の内容を変更するために使用できます。                                         |
| [![](attachments/819203/918009.png)](create-list)    | [リストを作成](create-list)   | リストの作成は、(空の) リスト変数を作成するために使用できます。                                      |
| [![](attachments/819203/917792.png)](list-operation) | [リスト操作](list-operation) | リスト操作は、同じエンティティのオブジェクトと2つのリストを結合または比較するために使用できます。                      |

## アクション通話アクティビティ

アクションコールアクティビティは、別のマイクロフローを呼び出したり、Java アクションを呼び出したりするために使用できます。

| グラフィック                                                 | 名前                                | 説明                                                                                 |
| ------------------------------------------------------ | --------------------------------- | ---------------------------------------------------------------------------------- |
| [![](attachments/819203/918018.png)](java-action-call) | [Java アクションコール](java-action-call) | Java Action コールを使用して Java アクションを呼び出すことができます。 引数はアクションに渡すことができ、結果は変数に格納することができます。   |
| [![](attachments/819203/918001.png)](microflow-call)   | [マイクロフロー通話](microflow-call)       | マイクロフロー呼び出しは、別のマイクロフローを呼び出すために使用することができます。 引数をマイクロフローに渡すことができ、その結果を変数に格納することができます。 |

## 可変アクティビティ

変数アクティビティは、マイクロフロー内で変数を作成または変更するために使用できます。

| グラフィック                                                | 名前                       | 説明                                     |
| ----------------------------------------------------- | ------------------------ | -------------------------------------- |
| [![](attachments/819203/918011.png)](change-variable) | [変数の変更](change-variable) | 変数は変数の値を変更するために使用することができます。            |
| [![](attachments/819203/918110.png)](create-variable) | [変数を作成](create-variable) | Create variable は、新しい変数を作成するために使用できます。 |

## クライアントのアクティビティ

クライアントアクティビティは、アプリケーションのWebクライアントにアクションを実行させるために使用できます。 例えば、別のページを表示したり、ファイルをダウンロードしたりしています。

| グラフィック                                                    | 名前                                | 説明                                                                                                                          |
| --------------------------------------------------------- | --------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| [![](attachments/819203/918114.png)](close-page)          | [ページを閉じる](close-page)             | 閉じるページは、このアクティビティが使用されているマイクロフローを呼び出すユーザーが最後に開いたページを閉じます。                                                                   |
| [![](attachments/819203/918108.png)](download-file)       | [ファイルをダウンロード](download-file)      | ダウンロードファイルは、ブラウザが特定のファイルをダウンロードできるようにするために使用できます。 このアクティビティが使用されているマイクロフローを呼び出すユーザーは、ダウンロードポップアップを取得したり、ファイルがブラウザに直接表示されます。 |
| [![](attachments/819203/918099.png)](show-message)        | [メッセージを表示](show-message)          | Show message を使用して、このアクティビティが使用されるマイクロフローを呼び出すユーザーへのブロッキングまたはノンブロッキングメッセージを表示できます。                                          |
| [![](attachments/819203/917544.png)](show-page)           | [ページを表示](show-page)               | Show page は、このアクティビティが使用されるマイクロフローを呼び出すユーザーにページを表示するために使用できます。                                                              |
| [![](attachments/819203/918097.png)](validation-feedback) | [検証のフィードバック](validation-feedback) | 検証フィードバックは、属性または関連性を表示するウィジェットの下に赤いテキストを表示するために使用できます。                                                                      |
| ![](attachments/16713769/17661963.png)                    | [ホームページを表示](show-home-page)       | Show home Page can be used to navigate the home page for the current user.                                                  |
