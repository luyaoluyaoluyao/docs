select ボタンは、参照セレクタまたは参照セットセレクタのオブジェクトを選択する際に使用されるグリッドの行の選択を確認します。 このため、 select ボタンは、参照セレクタまたは参照セットセレクタの 'Add' ボタンに接続されているページ内のグリッドにのみ配置できます。
